# PS3 Rich Presence for Discord - Release 2.1.2

## What's Changed in v2.1.2

- **Improved Presence Text Formatting**: When `"Use App / Game Name as Presence Title"` is enabled, line 2 now displays `"on PlayStation 3"` instead of repeating the game name. This prevents the game name from being duplicated across line 1 and line 2 in Discord.

## Download Assets
- `PS3RPD_GUI.exe` - Windows standalone GUI executable
- `PS3RPD.exe` - Windows standalone CLI executable
- `PS3RPD.py` - Standalone CLI Python script
- `PS3RPD-GUI-v2.1.2-Python.zip` - Complete Python source archive
